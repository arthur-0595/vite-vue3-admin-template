# vite-vue3-template

vite搭建的vue3后台模板，UI框架antdv 2.x

  # 安装依赖

  yarn

  # 启动服务

  yarn dev

### 项目目录划分

    - node-modules
    - public         公共资源
    - src 开发主文件
        - api        项目接口api，按模块划分文件
        - assets     静态资源，图片等
        - components 公共组件
        - icons      图标库，svg文件
        - layout     项目外层架构相关代码
        - plugins    插件
        - router     路由
        - store      全局状态
        - styles     全局样式
        - utils      公共方法
        - views      项目页面代码
